# Supported Units Reference

## Length
- m (meters)
- ft (feet)
- cm (centimeters)
- in (inches)
- km (kilometers)
- mi (miles)

## Weight
- kg (kilograms)
- lb (pounds)
- g (grams)
- oz (ounces)

## Temperature
- C (Celsius)
- F (Fahrenheit)
- K (Kelvin)

**Note**: Temperature uses a special formula (not simple multiplication). The script handles this automatically.
